Phase 1: Foundations & Chapter 2

Overview

This week focuses on expanding and deepening your literature review beyond what you accomplished in EDL 830. You'll learn advanced search strategies, develop systematic documentation protocols, and begin building the comprehensive literature foundation required for Chapter 2.

Learning Objectives

By the end of this week, you will be able to:

1
Conduct advanced database searches using Boolean operators
2
Develop systematic search documentation protocols
3
Identify seminal works and trace citation networks
4
Evaluate source quality using doctoral-level criteria
5
Organize literature using synthesis matrices

Independent Work Tasks

Complete these tasks and check them off as you go:

Develop Search Strategy

Create a comprehensive search strategy document including key terms, synonyms, Boolean combinations, and database selection rationale.

Conduct Systematic Searches

Execute searches across ERIC, ProQuest, JSTOR, and Google Scholar. Document results systematically.

Trace Citation Networks

Identify 3-5 seminal works and trace their citation networks forward and backward.

Set Up Reference Management

Organize all sources in Zotero, Mendeley, or EndNote with consistent tagging.
